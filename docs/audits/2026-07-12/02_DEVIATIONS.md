# 02 — Deviations

Audited against **Ka0s WoW Addon Standard v1.0.0 (2026-07-12)**. Prefix `LH-` (first audit). IDs are stable; a recurring deviation keeps its ID on re-audit. Evidence for every row is in `03_EVIDENCE.md`; remediation in `04_TECHNICAL_DESIGN.md` / `05_EXECUTION_PLAN.md`.

**Summary:** MUST failures **9**, SHOULD failures **2**. Verdict: **minor deviations** — the addon is architecturally strong (bus, schema, §6A window, debug console, tests all compliant); every open item is a packaging / documentation / panel-polish gap, none affecting core capture or storage.
